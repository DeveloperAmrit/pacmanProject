# Pady's Pacman

Pady's Pacman is a desktop game where players navigate a yellow circular character (Pacman) through a maze, eating pellets and cherries for bonus points while avoiding intelligent ghosts. The game features six unique maps, challenging gameplay with only one life, and a high-score tracking system.

## Features
- **Multiple Maps**: Explore six maps of varying sizes.
- **Intelligent Ghosts**: Challenging AI to test your skills.
- **High Score Tracking**: Stores and displays high scores for each map.
- **Thrilling Gameplay**: Bonus cherries and one life add to the excitement.

## Requirements
- **Visual Studio** (for development)
- **SDL2**, **SDL2_image**, and **SDL2_ttf** libraries
- **GCC 64-bit compiler**

## Tech Stack
- **C Language**
- **SDL2 Library**
- **Breadth-First Search (BFS)** algorithm for ghost pathfinding
- **Functions, Pointers, Structures, File Handling, and Header Files**


## How to Run the Game

### Step 1: Install SDL2 and Visual Studio
- Download and install SDL2 from [libsdl.org](https://www.libsdl.org/).
- Install Visual Studio with support for C and C++ development.

### Step 2: Create a Visual Studio Project
- Open Visual Studio.
- Create an **Empty C++ Project**. (Note: The project supports C, and this game is written entirely in C.)

### Step 3: Configure SDL2 with Visual Studio
- Connect SDL2 by following the setup instructions for Visual Studio:
  - Include SDL2 headers in the project.
  - Link SDL2, SDL2_image, and SDL2_ttf libraries.

### Step 4: Set Up the Project Folder
1. Right-click on the solution name in Visual Studio and select **Open in File Explorer**.
2. Copy all files and folders from the `Code` directory into the project directory opened in the previous step.

### Step 5: Build and Run
- Build the project in Visual Studio.
- Run the executable to start the game.

### Step 6: Explore and Enjoy
- Use keyboard controls to navigate Pacman.
- Click buttons to select maps, explore features, and enjoy the game.

## Screens
- **Home Screen**: Start the game or view high scores.
- **Select Map**: Choose from six unique maps.
- **Game Screen**: Play and survive the maze.
- **Game Over Screen**: View results and return to the home screen.
- **High Scores**: Check the high scores for each map.
